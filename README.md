# rvdecoder-ts

A simple decoder for RISC-V, written in TypeScript.

Demo: <https://ngc7331.github.io/rvdecoder-ts/>

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
